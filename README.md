# DevOps Git Lab

This repository contains hands-on Git practice exercises covering:

- Repository initialization
- Cloning
- Staging and commits
- Branching
- Merging
- Merge conflict resolution
- Rebasing
- Stashing
- Tagging
- Git workflows

Tools Used:
- Git
- GitHub
- Linux (Ubuntu)

Purpose:
To gain practical experience with version control as part of DevOps preparation.
